# golem-cmd

A command line interface for the golem R package.

```bash
npm install golem-cmd
```

## Usage

Create an app.

```bash
golem-cmd create myapp
```

Add a module (from root of golem app)

```bash
golem-cmd module module_name
```

Start app (from root of golem app)

```bash
golem-cmd start
```
